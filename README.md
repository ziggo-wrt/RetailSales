# :card_index_dividers: RetailSales - Data Analytics
**_RetailSales_** is a data analytics project focused on analyzing retail sales data using Microsoft Excel. The project demonstrates the full workflow of data analysis, including data cleaning, exploration, visualization, and generating business insights from sales data.

## :bulb: Overview
This project was created to:
- Showcases data analytics skills using Microsoft Excel
- Analyze customer purchasing behavior and sales trends
- Generate actionable insights to support business decision-making

## :brain: Objective
The goal of this project are to:
- Identify sales trends over time
- Analyze purchasing behavior based on age, gender, and categories
- Explore the relationship between unit price and customer age groups

## :gear: Tools & Technologies
- `Microsoft Excel` for data analysis and charts
- `Microsoft OneDrive` for storing data analysis files

## :jigsaw: Getting Started
1. Click the links in this repository into OneDrive
2. Review the data, analysis sheets, and dashboards

## :mag: Discoveries & Insights
Some key findings from the analysis include:
- While buying habits are similar across ages 18-64, women outpace men in the 26-42 age group
- Product categorization is independent of price points, as categories exist across all price levels
